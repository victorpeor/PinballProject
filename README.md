# PinballProject

## My Little Pony Crystal Princess: The Runaway Rainbow
This game is an adaption of the Game Boy Advance, My Little Pony Crystal Princess: The Runaway Rainbow. Here's a link of the original gameplay https://youtu.be/7OpbqyG5-PY?si=GXduMBWfPIKgSrKm&t=2301 

This game was made with Raylib, Box2D and C++.

### Features 
Right Arrow: Control right flipper.

Left Arrow: Control left flipper.

Down Arrow: Controlling the spring.

Space: Restart the game when appears the game over screen. 

F1: Show collisions.

F2: Spawn new ball.
